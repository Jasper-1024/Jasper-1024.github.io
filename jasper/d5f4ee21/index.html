初入树莓派1安装系统+进入SSH

导语

为了更好的从事程序员事业、填坑linux、为学弟打下第一手资料、留下linux培训教材……

  • 实际就是为了搭个路由器，玩玩
    入了树莓派3b +电源+外壳+sd卡—300大洋--------肉疼

  • 参考

树莓派系统安装

  • 树莓派3b的耗电量还是挺大的
    推荐买个2.5A的电源

系统选择

  • 树莓派支持 数十款操作系统，一般推荐的是 raspbian .

  • Raspbian是为树莓派设计，基于Debian的操作系统，不隶属于树莓派基金会[1]，但被列为官方支持的操作系统，

  • Raspbian镜像

  • Raspbian有三个版本，简单区别如下

    • Raspbian Buster with desktop
      带有完整的桌面体验，安装包解压后 3.56G 左右,新手推荐.
    • Raspbian Buster Lite
      没有X-server 套件，没有 Qt 和 GTK+ 的函式库。等于去掉了桌面.非常轻量. 3B 启动后占用不到 100M.
    • Raspbian Buster with desktop and recommended software
      带有完整桌面体验,和官方推荐的软件,更大更全.

  • 所要用到的工具是：

    1. 多功能读卡器一个
    2. TF卡一张（4G以上,现在镜像都挺大,推荐 16G 起步）

  • 烧写器官方推荐是win32diskimager
    但是文件路径不支持中文。

  • 个人推荐 USB Image Tool
    故障率最低的是1.7版本。

  • (后记,随便一个写盘的就行,现用 rufus)

  • ps : Kali 对不同网卡支持更好,还有 64 位支持,刷入过程相同.

插上读卡器，烧写系统

  • 烧写系统

    1. 打开USB Image Tool
      注意左上角是Device Mode，不是Volume Mode！
    2. 选择下载后解压的img文件
    3. 写入等待进度条完成即可。

  • 错误原因

    1. USB Image Tool 在win7以上系统中必须要以管理员权限运行
    2. USB Image Tool运行需要.net环境 直接使用系统默认加载的即可。
    3. 烧写完成后接入电源
    4. 或者换 rufus .

进入SSH管理（没有外接显示屏时）

  • Raspbian 默认关闭了远程ssh.

  • 在sd卡的 根目录 新建一个空文件,文件名 ssh ,插入树莓派,上电.

  • 第一次进入系统时直接利用网线接入路由器，将电脑接入同一个wifi下。
    在路由器管理界面查找树莓派对应得IP地址。
    也可以扫描一下局域网设备中开启22端口的设备.

  • 使用SSH管理工具，这里以putty为例。打开软件，填入树莓派ip地址。

    1. 打开软件，填入树莓派ip地址。

    2. 默认用户名pi 密码raspberry。

    3. 登陆成功！

  • 不愿再用第三方 SSH 客户端的,现在 win10 自带了 ssh .

    输入 yes 记住这个 ssh ,之后输入 raspberry 默认密码.建议登录后修改默认密码.
