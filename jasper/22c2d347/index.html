知网 pdf 下载

有些文献特别是学位论文知网只能下 caj,奈何还是 pdf 兼容性好.

参考这里 > https://www.zhihu.com/question/25275044

改 url

  • 下载链接种的 cajdown -> pdfdown .有可能需要改两遍.
  • 没有 cajdown,转而搜索 &dflag=nhdown -> &dflag=pdfdown (nh 改成 pdf)

或者 知网海外版 直接下载 pdf…

这样下的 pdf 可能没有目录,智能目录自行搜索了.

pdf 自动书签

知网海外版 下载的 pdf 不带书签,看的时候可麻烦死了.

PDF 补丁丁 可以比较自动化的完成自动书签.

只针对非扫描版,且章节标题有字体/大小区分.

PDF 补丁丁 0.6.1.3498 版重大更新：为文本PDF文档自动生成书签！

CCF等级标注

CS 下 CCF 等级是绕不开的事情.

CCFrank4dblp 是个浏览器插件,可以在常用文献搜集软件显示 CCF 等级.

参考文献导入

写相关研究经常是, 文献 ⇌\rightleftharpoons⇌ 参考文献 的无限循环.写到最后,简而言之手疼…

此时有两个可以处理参考文献,一次性导出到 zotero 的网站.

Reference Extractor

  • 上传 pdf/doc 等解析引用文献,导出到 zotero.
  • 只能处理由 zotero 生成的引用,直接用 word 编辑的引用暂时无能为力.

citation-finder

  • 手动复制粘贴参考文献到网站,开始解析,最后导出.
  • 原理是重新搜索粘贴的参考文献,最好挨个比较.
  • 对英文很准确,但是中文免谈.

Zotero

开源,免费,强大的文献管理软件.

插件

  • zotero-better-bibtex: 可以导出 bib 格式文件,并一直随着文献库更新,务必导出时选择 keep updated
  • asminum: 中文名茉莉花… Zotero 对直接导入的英文 pdf 识别甚佳,中文则不咋地,这个插件完美解决了问题.茉莉花自带了很多中文的文献插件(??) 记着点更新…
    • 同时别忘了更新浏览器插件 --> Zotero 更新 知网Translator 翻译器教程

自动文献下载

参考> zotero-shortdoi + Sci-Hub，让99%的文献都能被免费下载!

起缩写名称

英文非母语,起算法的缩写名称真的是折腾 -> acronymify 输入全名,会尝试生成缩写,能够提供一定参考.

软件学报参考文献格式

参考文献自然是全部托管到了 zotero ,最近疯狂肝稿子,被告知按照 软件学报 模板调整格式…但是 zotero 没有这个参考文献格式啊…

最终还是找到了 -> journal-of-software.csl 导入zotero 完美…

zotero 暗色主题

zotero-night 主界面和 pdf 阅读覆盖,用到很舒服,记着给作者点个 star .

GB-T-7714 2005 的坑坑坑

没想到最后居然栽倒这里了, zotero 的 GB-T-7714 2005 有若干 bug 包括但不限于

  • 作者超过 3 人英文期刊也有等号.
  • 比参考样式, 多出来 DOI 链接等等
  • 作者名字首字母小写xxx
  • xxx

严格来说这个锅不在 zotero 而是学校的奇葩规定导致的, 何况这个标准本来就是 中文标准,但是无奈脸.

感谢 Chinese-STD-GB-T-7714-related-csl 这个开源项目,我能找到最接近的是 420beihang-university.csl 北京航天航空大学的格式,以此为基础稍加更改,即可得到完整可用参考文献格式.
