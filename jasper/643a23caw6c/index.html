导语

一直会听到的论文等级

  • A B C 等级
  • sci 一区二区
  • 中文核心/北大核心/南大核心

头大,这里总结一下.

论文等级

核心/非核心

  • 是按照某个筛选体系,分为核心/非核心.

分类

  • 北大核心: 以自然科学为主
  • 南大核心: 以社会科学为主
  • 中国科技核心期刊: 中国科学技术信息研究所
  • CSCD 中国科学引文数据库来源期刊: 中国科学院文献情报中心

SCI/EI 等

sci / ei /istp 是国际上被国内认可的检索体系.

  • sci 是美国科学引文索引.
  • ei 是美国工程索引.
  • istp 是美国科技会议录索引.

sci 再往下是 sci 分区,有两种.

  • JCR 分区

    • 176个学科 4个区.
    • 25% Q1
    • 25%~50% Q2
    • 50%~75% Q3
    • 其他 Q4

  • 中科院分区(计算机用的较多)

    • 13 类 4 个区
    • 前 5% 1区
    • 6%~20% 2区
    • 21%~50% 3区
    • 其他 4区

A B C 类

这个是计算机类特有的,是中国计算机学会 (ccf) 推荐的高水平期刊/会议.

  • 有 A B C 三类.
  • 但并未包含全部期刊,未被收录的期刊不一定质量不行.

具体参考 中国计算机学会推荐国际学术会议和期刊目录-中国计算机学会

计算机类还是参考 ccf 等级多一点.

总结

上面大概是现阶段遇到的论文等级,其他内容有时间再补充.

搜索论文搜索的太烦了,有时间写成 python 爬虫才好.
