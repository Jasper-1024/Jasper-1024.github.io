LAB2

lab2 主要是 2 个部分:

  • Wrap
  • TCPReceiver

Wrapping

wrap: 3 种值的转换, seqno/ abs seqno / stram index;

  • seqno 32bit, 取模循环, ISN 开始; abs seqno 64bit 从 0 开始; strean index, 64bit, 不包括 syn fin.

warp 就是包装 seqno 在 3 种值之间转换 seqno≡ISN+absolute_seqno(mod232)\text{seqno} \equiv \text{ISN} + \text{absolute\_seqno} \pmod{2^{32}}seqno≡ISN+absolute_seqno(mod232)

  • 公式推导不再赘述

go 没有重载, 创建了 WrappingInt32 类型, 为类型实现 warp / Unwrap 方法.

  • 位操作, 与 c 基本相同, 区别有: 取反 ~© 和 ^(go); go 增加了 &^ (AND NOT), 位清除掩码;

TCPReceiver

recv 的逻辑与编程语言关系不大, 逻辑细节: syn fin 处理, syn 带不带数据等.

SegmentReceived:

  • 还未收到 syn 拒绝任何 seg
  • 计算 checkpoint 和 abs seqno
    • 如果没有 syn 且 abs seqno == 0, 则 seg 可能是重复 syn 停止处理.
  • stream index 和 fin 的处理, date 写入重组.~~~~
